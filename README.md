# Campus Care 🎓

Campus Care is a comprehensive, modern College Management System designed to bridge the gap between administration, faculty, and students. By digitizing essential campus activities into a centralized, real-time dashboard, Campus Care ensures transparent communication and efficient academic management.

## 📌 Problem Statement
Traditional college management relies heavily on fragmented systems, manual data entry, and disparate communication channels. Students struggle to track their live attendance, placement drives, and leave approvals. Faculty members spend excessive time manually marking attendance and managing schedules. Administrators lack a unified birds-eye view of campus activities, resulting in administrative bottlenecks and a lack of actionable analytics.

## 💡 The Solution
Campus Care provides a unified, multi-role web application with tailored dashboards for every stakeholder in the ecosystem. With a soft-pastel, high-density design system, the platform prioritizes user experience while handling complex relational data. 

### Key Stakeholder Roles
1. **Admin**: Oversee the entire system, manage user accounts, classes, and view high-level analytics.
2. **Student**: Track daily attendance, apply for leaves, view live placement opportunities, and monitor mess/transport schedules.
3. **Class Teacher**: Manage the full roster for a specific class, handle student leave requests, and monitor overall class performance.
4. **Subject Teacher**: Record subject-wise attendance and manage daily lecture schedules.
5. **Placement Cell**: Track student placement applications, manage upcoming drives, and publish active job opportunities.
6. **Mess Owner**: Publish weekly menus, track student meal subscriptions, and gather feedback.

## 🚀 Key Features
* **Real-time Attendance Tracking**: Instant updates for students, complete with dynamic charts (Recharts) and leave-bonus calculations.
* **Leave Management System**: Streamlined request and approval workflow for students and class teachers.
* **Placement Hub**: Dedicated portal for tracking active companies, upcoming recruitment drives, and student selection statuses.
* **Live Notifications**: Powered by Socket.io for instant updates regarding events, placements, and important campus announcements.
* **Modern UI/UX**: Built with React and Tailwind CSS, featuring a responsive, soft-pastel aesthetic with micro-animations and data-dense table layouts.

## 🛠️ Technology Stack
**Frontend:**
* React (Vite)
* Tailwind CSS (Styling & Design System)
* Recharts (Data Visualization)
* Lucide React (Icons)
* Axios (API Client)

**Backend:**
* Node.js & Express.js
* MySQL (Relational Database)
* Socket.io (WebSockets for real-time features)
* JWT (Role-based Authentication)

## 🎯 Conclusion
Campus Care transforms the disjointed college experience into a streamlined, interactive digital campus. By automating routine administrative tasks and providing real-time data visibility to all stakeholders, the platform allows institutions to focus on what truly matters: education and student success.
